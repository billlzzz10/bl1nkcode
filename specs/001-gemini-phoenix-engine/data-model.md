# Data Model

## Semantic Index (Milvus)

Collection Name: `phoenix_index`

| Field | Type | Description |
|-------|------|-------------|
| `id` | VarChar (PK) | Unique identifier for the item (e.g., `plugin:name:command`). |
| `vector` | FloatVector(768) | Embedding vector generated by `nomic-embed-text-v1.5`. |
| `text` | VarChar | The searchable text content (description, usage, etc.). |
| `type` | VarChar | Category of the item: `command`, `skill`, `documentation`, `persona`. |
| `source` | VarChar | Origin of the item (e.g., `catalog`, `user-docs`). |
| `payload` | JSON | Full definition or reference data required for retrieval. |

## Catalog Structure (`catalog/index.json`)

Source of truth for all available capabilities.

```json
{
  "metadata": {
    "version": "1.0.0",
    "updated_at": "ISO8601-DATE"
  },
  "plugins": [
    {
      "name": "string",
      "description": "string",
      "version": "semver",
      "commands": [
        {
          "name": "string",
          "description": "string",
          "args": [
            { "name": "string", "type": "string", "required": boolean }
          ]
        }
      ],
      "skills": [
        { "name": "string", "path": "path/to/skill" }
      ]
    }
  ]
}
```

## Agent Persona (`agent-personas/*.toml`)

Configuration for specialized agents.

```toml
name = "string"
description = "string"
version = "string"

[models]
preferred = "gemini-pro"

[skills]
include = ["skill-name-1", "skill-name-2"]

[prompts]
system = "path/to/system/prompt.md"
```
